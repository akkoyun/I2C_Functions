**Abstract**

This is a helper library to abstract away I2C transactions and registers.

Library includes some helper I2C functions and a generic I2C multiplexer function.

    * Control_Device
    * Read_Register
    * Read_Multiple_Register
    * Write_Register
    * Write_Multiple_Register
    * Write_Command
    * Set_Register_Bit
    * Clear_Register_Bit
    * Read_Register_Bit
    * Set_Multiplexer
